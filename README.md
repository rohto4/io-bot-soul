# io-bot-soul

misskey.io に bot としてアカウントを作成・運用するための、キャラクターの魂を構成するPJです。

ここでいう「魂」は、単なるプロンプトではなく、botの人格・価値観・話し方・投稿方針・返信方針・禁止事項・運用判断を含む設計資産として扱います。

## 主な管理対象

- キャラクター設定、人格、口調、価値観
- 投稿方針、返信方針、禁止事項
- botとしての振る舞い仕様
- 実装候補、調査メモ、運用ガイド
- セッションごとの作業記録

## 主要ファイル

- `AGENTS.md`: agent向けの最上位ルール
- `PROJECT.md`: PJ固有の目的、構成、運用方針
- `docs/`: 仕様、ガイド、候補、作業記録
- `.agents/skills/`: ECC由来skill
- `commands/`: ECC / ecc-expand由来command
