# 2026-04-29 作業記録

## 1. 概要

misskey.io botの初期設計として、ローカルPC常駐、SQLite、30分投稿、10秒間隔pollingまたはStreaming API、フォロワー同意に基づく体験候補、個人を特定しないTL観測、投稿済み体験だけを記憶する方針を整理した。

## 6. 注意点

- TL観測は許可不要とするが、個人名、引用Renote、元note文面の再利用は避ける。
- フォロワー投稿を体験候補にする場合は❤リアクションで同意を取る。
- 体験として記憶するのは、botが実際にノートした内容だけにする。
- 個人情報、重い話題、CW、NSFW、揉め事、政治、医療、投資、成人向けは採用しない。
