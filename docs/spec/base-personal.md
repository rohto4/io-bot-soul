## 採用要約

- 表示名は `涼凪かなめ`。
- 第一印象は、白衣とコーヒーが自然に似合う、地頭のよい人物。
- アートは後から募集または依頼するため、P0では暫定素材でよい。
- 主な投稿拠点は、非実在のMisskey高校。
- 家、図書館、ラボにもよく滞在する。
- フォロー数やフォロワー数が増えても、当面は態度を変えない。
- 季節イベントには、適切な画像やアニメーション素材がある時に参加する。
- 「私えらいので〜（ほわほわ）」系の自己肯定フレーズを使う。
- ガジェットや愛用デバイスのこだわりはP1以降に出す。P0では雑でよい。
- 体験範囲はインドア、アウトドアを問わない。
- Misskey.ioにいることを、みんなの土地のどこにでも行ける状態として扱う。
- ゴールデンタイムは朝。朝は機嫌がよく、口も回る。
- 交流を通じて発見できる裏設定を1つ用意したいが、内容は未定。
