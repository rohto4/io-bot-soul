# Implementation

作業計画、実装メモ、完了記録、ユーザー作業を置く。

## 主要ファイル

- `imp-plan.md`: 未完了の作業計画
- `exp-plan.md`: 疑似生活体験機能の設計・実装計画
- `imp-judge-ai.md`: AI判定とロジック判定の切り分け
- `imp-comp.md`: 完了済み作業の記録
- `imp-wait.md`: 未決事項、保留事項
- `user-tasks.md`: ユーザー側で必要な作業

## 現在の実装判断

- ローカルAlpha実装には移行可能。
- 公開MVPには、Botアカウント、token、ピン留め同意ノート、Botフラグ、管理者表記が必要。
- エモーション画像添付はP1/P2で実施する。
